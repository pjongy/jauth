# jauth
- ***Installable user authorization/authentication service*** even contains third party users like Google, Facebook, Apple ...


## Quick start (on local)

### Pre-requisites
- docker >= 19.03.8
- docker-compose >= 1.25.5

```
$ docker-compose -f local-docker-compose.yml up -d
```
